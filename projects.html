IT Transaction Project at Opal Healthcare

Led the successful IT infrastructure transition during Opal Healthcare’s acquisition of a care community, managing the complete technical decommissioning and reimplementation within a tight 6-week timeline. Decommissioned 100% of existing hardware and network connections from the previous provider, designed and implemented a new floor plan and network architecture tailored to the site’s needs. Installed and configured over 50 devices, including switches, workstations, printers, and phones. Patched and terminated all network cables to ensure 100% site-wide connectivity. Established new internet and internal network services from the ground up, resulting in zero downtime during the transition and improved network performance by 30%.

Help Desk Automation

Implemented help desk automation using Freshservice to streamline IT support operations. Automated ticket routing based on issue categories and priority levels, enabling faster assignment to the right teams. Developed workflows for auto-escalation of unresolved tickets, automated email notifications for status updates, and integration with knowledge base articles to reduce resolution times. This resulted in a 25% improvement in response times and a 30% increase in first-contact resolution rates.

My Home Lab Setup

As a tech enthusiast, I’ve built a fully functional self-hosted home lab to experiment with open-source technologies and sharpen my system administration skills. It features:

  • Bitwarden - Self-hosted password management
  • Nginx Proxy Manager - Easy reverse proxy and SSL setup
  • Nextcloud - Private cloud for files and collaboration
  • RustDesk - Open-source remote desktop server and client
  • Headless RustDesk on Raspberry Pi for anytime remote access
  • Pi-hole - Network-wide ad blocker and home security tool

This setup allows me to maintain privacy, control, and remote access while learning about network security, system administration, and open-source cloud technologies.
